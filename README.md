## How to set up 


Edit your `settings.json` file before starting your bot.
```json
{
    "owner": {
        "id": "BOT'S OWNER ID"
    },
    
    "bot_token": "YOUR TOKEN",

    "database": {
        "host": "DATABASE HOST",
        "user": "DATABASE USER",
        "password": "DATABASE PASSWORD",
        "name": "DATABASE NAME"
    }
}
```
